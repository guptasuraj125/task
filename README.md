My Frontend Design Project

This project is made using Next.js and Tailwind CSS.

It is only frontend design. No backend is added.
All pages are working with simple mock navigation.

Tech Used

Next.js

Tailwind CSS

TypeScript

Zod or form validation

How to Run

Install packages:

npm install

Run the project:

npm run dev

Then open:

http://localhost:3000

Project Structure

app → all pages

components → reusable components

hooks → custom hooks

validation → form validation schemas

types → TypeScript types

utils → helper functions
